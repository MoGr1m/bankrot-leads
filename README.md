# Bankrot Leads — агрегатор лидов для юрфирм по банкротству физлиц

## Идея

SEO-агрегатор по модели "рестоклаб / профи.ру": парсим юрфирмы, специализирующиеся
на банкротстве физических лиц, создаём страницы городов/услуг/юристов, привлекаем
органический трафик из поисковиков, собираем заявки от людей с долгами и продаём
эти заявки (лиды) юрфирмам.

Ключевой дифференциатор от конкурентов (Правовед.ру, 9111.ru): используем
**реальную судебную статистику** из открытых государственных реестров
(картотека арбитражных дел, ЕФРСБ) для рейтингов финансовых управляющих —
такой контент сложно скопировать и он вызывает больше доверия, чем накрученные отзывы.

## Модель монетизации

Лид (человек, оставивший заявку с суммой долга и контактами) продаётся
юрфирмам по 3000–10000₽, в зависимости от города и качества лида
(валидируем по сумме долга и наличию дохода перед продажей).

## Источники данных

- **2ГИС Каталог API** — список юрфирм по банкротству в конкретном городе.
  См. `src/parsers/parse_2gis.py`.
- **kad.arbitr.ru** (картотека арбитражных дел) — статистика по делам о банкротстве.
  ⚠️ Требует проверки: есть ли открытый API или нужен веб-скрапинг с обходом
  антибот-защиты. Пока не реализовано.
- **fedresurs.ru** (ЕФРСБ) — реестр сведений о банкротстве, финансовые управляющие.
  ⚠️ Аналогично требует проверки перед реализацией парсера.
