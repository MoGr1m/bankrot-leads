Сайт bankrot-leads — что есть и как пользоваться

Что есть на сайте

Главная

Что такое банкротство, как всё работает, выбор из 5 городов (Москва, СПб, Челябинск, Пермь, Воронеж), FAQ. Две кнопки: «Пройти подбор» и «Выбрать город».

Подбор компании /podbor-kompanii

Ядро сайта.

  • Квиз из 7 вопросов: город, сумма и виды долгов, просрочка, доход, имущество, сделки.
  • По ответам показываются 3–5 компаний с объяснением, почему подходят.
  • Пользователь отмечает до 3 компаний, оставляет телефон → заявка.

Каталог города /chelyabinsk

Список юрфирм с фильтрами (проверенные / с офисом) и сортировкой. У каждой — значок проверки и показатель уверенности вместо звёзд.

Карточка компании

Реквизиты (ИНН/ОГРН, если найдены), таблица «что проверено» с источниками, дисклеймеры, своя форма заявки.

Услуги · Ситуации · Статьи

SEO-страницы: судебное и внесудебное банкротство, коллекторы, приставы, ипотека, плюс калькулятор стоимости (показывает структуру затрат, без выдуманных сумм).

Прозрачность подвал

Методология, как проверяем компании, политика отзывов, раскрытие о рекламе, политика обработки ПД.

Как пользоваться — заявки

Заявки приходят в Telegram, двух типов:

🆕 Заявка из подбора

Телефон + ситуация из квиза + выбранные компании.

Заявка с карточки

Телефон + название конкретной фирмы.

В каждой заявке есть строка-заготовка статуса — отмечайте вручную (новый / перезвонил / передал фирме).

По бизнес-логике: сначала копятся реальные лиды с сайта, потом холодный аутрич к юрфирмам — не параллельно.

Как обновлять данные

Добавить или обновить компании в городе:

# перейти в проект
cd /Users/mogr1m/Projects/bankrot-leads

# спарсить город из 2ГИС
python3 src/parsers/parse_2gis.py

# добавить ИНН/ОГРН из ЕГРЮЛ
python3 src/parsers/enrich_dadata.py
⚠️ Важно: после парсинга обязательно закоммитить и запушить data/*.json — иначе на проде останутся старые данные (сайт читает JSON при сборке).

Деплой — автоматический: любой git push в ветку main → Vercel сам пересобирает и публикует.

Что сайт честно НЕ обещает

Нет гарантий списания долгов, нет выдуманных цен и рейтингов. Где данных о компании нет — прямо написано «не проверено». Это сознательно: требование ТЗ и ФЗ «О рекламе».
